# MaYi's React App boilerplate

MaYi's React App boilerplate - A simple template repo help to build MaYi-style website.


### Prerequisites

```
yarn or npm
```

### Installing

1. `yarn install` or `npm install`
2. run dev `npm run dev`
3. build the final page `npm run build`
